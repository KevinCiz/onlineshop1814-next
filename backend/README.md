# Setup

## Environment Variable
Copy **.env-template** & Fill up the empty value
1.  NODE_ENV
	* development / production
2. MONGO_DB
	* Mongo db URL
3. PORT
	* backend port (current demo: 5001)
4. APP_SECRET_KEY
	* secret sign key for authentication
5. CLIENT_URL
	* Front end URL (current demo: http://localhost:3000/)
